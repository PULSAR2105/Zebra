## Getting Started

### Prerequisites

A compiler for NASM and a link editor. Normally, everything is available in the repository.

* Windows :
  ```
  compiler : nasm
  linker : GoLink.exe
  ```
* Linux :
  ```
  compiler : nasm
  linker : ld
  ```
